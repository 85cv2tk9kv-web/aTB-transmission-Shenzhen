# Population-based Genomic Evidence for Hidden Transmission of Asymptomatic Tuberculosis in Urban China

This repository contains code and configuration files used for statistical analyses, genomic clustering, phylogenetic inference, and transmission reconstruction in the study.

## Repository structure

- statistical_analysis/
  Statistical analyses performed in R.

- genomic_analysis/
  SNP distance calculations and genomic clustering procedures.

- BEAST/
  BEAST XML configuration files and downstream phylogenetic processing scripts.

- BREATH/
  Scripts used for Bayesian transmission inference and extraction of secondary case metrics.

- metadata_example/
  Example de-identified metadata for demonstrating input structure.

## Software versions

Analyses were performed using:

- R version 4.3.1
- BEAST version 2.7.7
- BREATH version 0.0.1


## Data availability

Whole-genome sequencing data are available from the NCBI Sequence Read Archive under accession number PRJCA046418.

Individual-level epidemiological data are not publicly available because of ethical and privacy restrictions.
